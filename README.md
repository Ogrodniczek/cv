# CV and termination LaTeX template
> Full toolkit needed to change a job

## Prerequisites
Packages required to build each component:
- make
- texlive-base

### cv.pdf
- texlive-latex-extra

### termination.pdf
- texlive-lang-polish

## Building
To build final PDF documents use GNU make:
```
make [all]
make cv.pdf
make termination.pdf
```
